可提供的职位

  • 产品设计部

    设计师

    基本要求：
    1. 熟悉Photoshop、Dreamweaver、Fireworks；
    2. 熟悉HTML/CSS，能根据不同的浏览器进行页面制作；
    3. 能准确把握需求，通过分析和讨论，协助交互设计师和产品经理完成相应的视觉设计工作；
    4. 有原画创作能力。

  • 客户端开发组

    网站前端开发

    基本要求：
    1. 熟悉HTML、CSS、JavaScript等前台相关技术，熟悉W3C网页标准，能手写符合规范的页面，使标签语义化，并处理浏览器间的兼容性问题；
    2. 理解AJAX运作机制，熟练运用至少一种AJAX框架，如jQuery、YUI、Dojo、ExtJS、Prototype等；
    3. 了解前沿Web开发技术，如响应式设计，Media Queries，HTML5/CSS3。

    手机端开发

    基本要求：
    1. 熟悉C++语言，学习能力强；
    2. 熟悉http/Socket编程；
    3. 熟悉cocos2d-x、lua者优先；
    4. 熟悉一般常用的2D、3D图形算法,熟练运用lua脚本语言,熟悉IOS/Android平台开发流程；
    5. 对面向对象的设计理念有较好的理解和运用。

    Flash开发

    基本要求：
    1. 熟练使用ActionScript3；
    2. 有良好的面向对象设计能力以及开发习惯；
    3. 熟悉javascript与jQuery者优先。

  • 系统开发组

    网站后端开发

    基本要求：
    1. 熟悉某种Web后端语言/框架，如PHP，Python，Ruby；Django，Ruby On Rails；
    2. 熟练掌握SQL语句；
    3. 熟悉MYSQL数据库的配置和优化；
    4. 具有SVN或GIT等版本控制系统在开发过程中的使用经验。

  • 市场部

    市场运营主管

    岗位职责：
    1. 深入了解网络游戏行业发展趋势及市场竞争态势，有效制定市场战略规划及市场营销方案；
    2. 结合公司发展战略，制定市场策略、广告策略、品牌策略和公关策略，并推进执行；
    3. 组织实施市场研究及市场分析，对市场的变化做出准确的判断和预测，对产品更新及公司决策提供有效建议；
    4. 有效利用市场手段帮助运营部门达成营销目标；
    5. 进行游戏市场调研与分析，研究同行、业界发展状况，定期进行市场预测及情报分析，为公司决策提供依据；
    6. 负责部门日常业务管理、人才培养，从业务能力、营销理念等各方面整体提升团队水平。

    任职要求：
    1. 本科及以上学历，市场营销类相关专业；
    2. 热爱游戏行业，并充满激情；
    3. 5年以上相关行业市场经验，3年以上市场团队管理经验，并有成功的营销案例；
    4. 具备良好的市场敏感度，对网络游戏行业的市场营销工作有深刻的理解和认知；
    5. 掌握丰富的平台、媒体资源，并保持良好的合作关系；
    6. 有大局观，具备优秀的沟通能力和出色的谈判技巧；
    7. 有较强的组织协调能力和团队合作意识。
